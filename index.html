HTML

Every webpage is built using these three basic building blocks - structure, design and function.

HTML is the structure, if forms the skeleton of the site. It is a fairly simple language consisting of tags - open tags such as <p></p> and self-closing tags such as <img>. Open tags are for areas of text or function that begin in one place and end where the coder tell it to, such as paragraphs; self-closing tags are for images or links, they have a set beginning and end.

<p>This is a paragraph, an open tag, it can go on forever unless we add specific break points. To do that we use the self-closing tag <br>
This is a new line after the break. </p>
CSS

It's rather ugly though, isn't it? Pure HTML uses the default background color and font, it positions everything minimally and is just generally pretty ugly. That's where our design building block comes in - CSS

CSS stands for 'Cascading Style Sheet', the name referring to its passive trikle down property where child elements acquire their parent's styling - for example if we give a certain section a blue background color, all paragraphs within this section will have the same blue background.

To do so we add style attribue within our HTML tag

<p style='background-color:blue'>This paragraph now has a blue background</p>

We can also add all of our styling within a separate page reserved for CSS styling only.

Javascript

Well that is nice, very nice really, but what can you do with just structure and design? HTML and CSS are like the bones and skin of the site, without the muscle it just sort of lies there, it's static. Javascript makes our website dynamic, so that if we click on a button to order some delicious dumplings the order will be sent to the restaurant and the money charged from our account, without Javascript we would still be clicking on the image in frustration, with no hope of dumplings coming our way.

How do we implement Javascript in our site? Let's make a button which, when clicked, greets the user by name:

                
                    <button onclick="helloUser()"> Click Me! </button>

                    <script>
                        function helloUser(){
                            userName = prompt('What is your name?');
                            alert(`Hello, ${userName}!`);
                        }
                    </script>

We use a script open tag, within which we add function functionName(){ our code }.
Just like with CSS, we can also write all of our code in a separate page, reserved for Javascript only. For both CSS and Javascript this is useful when we have a lot of styling and codind to do, since it can take up a lot of space.

Examples

So far we've seen some HTML, HTML+CSS and finally HTML+Javascript. Let's see an example of how to join them all together:

Pretty cool right? Here's how we do it:

  1. Create a button using HTML
  2. Give the button an id → as the name suggests, it's a way to identify this button from other buttons
  3. Add a basic background color using CSS
  4. Add a function that executes when the button is clicked → onClick
  5. Create our function using Javascript →
    1. Find our button using its id → getElementById
    2. Create a variable (or container) for the color we want to change to
    3. Find the color we have now → myButton.style.backgroundColor finds the current CSS value
    4. Choose the color to change to according to the current background color → switch is a conditional statement, we'll learn about these later
    5. Assign the new color to our button

                
                    <button style="background-color: white" id="myButton" onclick="changeColor()"> Click Me! </button>

                    <script>
                        function changeColor(){
                            const myButton = document.getElementById('myButton');
                            let newColor;
                            switch(myButton.style.backgroundColor){
                                case 'white':
                                    newColor = 'blue';
                                    break
                                
                                case 'blue':
                                    newColor = 'magenta';
                                    break

                                case 'magenta':
                                    newColor = 'orange';
                                    break

                                case 'orange':
                                    newColor = 'white';
                                    break
                            }
                            myButton.style.backgroundColor = newColor;
                        }   
                    </script>

That's it! We now have a button that changes color when we click on it, and we used some basic HTML, CSS and Javascript to build it!
